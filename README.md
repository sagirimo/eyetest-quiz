# 眼科学期末题库 - 刷题工具

深圳大学医学部眼科学期末考试题库，整理自三个学生小组的出题，共 **1064 道选择题**，31 位出题人。

## 快速开始

```bash
# 直接打开刷题
open quiz.html
```

或部署到任意静态服务器。

## 功能

- **解析模式**：做题后显示答案解析
- **速刷模式**：答对自动跳转下一题，答错停下
- **按出题人筛选**：按组别和出题人过滤题目
- **收藏/星标**：标记重点题目
- **键盘快捷键**：A-E 选择答案，Space 下一题
- **进度追踪**：localStorage 记录进度

## 文件说明

| 文件 | 说明 |
|------|------|
| `quiz.html` | 独立刷题页面（内嵌全部题目数据） |
| `questions.json` | 提取的题目数据（1064 题） |
| `build_quiz.py` | 从 questions.json 构建 quiz.html |
| `add_explanations.py` | 自动生成题目解析（899/1064 已覆盖） |
| `extract_v3.py` ~ `extract_v5.py` | 题目提取脚本 |
| `眼科出题第一组/` `第二组/` `第三组/` | 原始 Word/PDF 题目文件 |
